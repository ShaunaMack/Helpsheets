HTML Basics

  1. type <!DOCTYPE>

    2. Tells computer the type of document it is reading

  2. <html> at beginning and </html> at the end

    3. Write the html stuff in between here

  3. open and close head tags (<head> </head>)

    4. Write the stuff that will show in the tab in between here
    Also tell the computer to check out the style from the CSS file in between here using the link with relationship below
    <link href = "URL of the CSS doc" type= "text/css" rel="stylesheet"> self closing tag

  4. open and close body tags (<body> </body>)

    5. The body is where you write all the content that will display on your web page

Body elements

Text tags include:

<h6> </h6> Creates the smallest headline
<p> </p> Creates new paragraph
<br/> Creates line break
<b> </b> Creates bold text
<i> </i> Creates italic text
<em> </em> Emphasizes a word italic ?
